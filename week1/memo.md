### PR
- 완성이되면 PR을 올릴거고, 저번 홈랩 포맷을 따라가면 된다.
- 리포트에 대한 요약
- 자신 Repo가 따로 있다면 해당 리포링크도 클클에 복사하면 좋다
- 무조건 PR commit 안에는 문서 작성 템플렛 Report 문서가 있어야함 (최소)

### PR Review
- 랜덤으로 PR을 달도록 돌릴거다.
- 3명이 랜덤으로 PR담당자가 되어 리뷰를 해야하게된다.
